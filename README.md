#  Sistema de Gestión Metalúrgica

Aplicación web full-stack desarrollada como Trabajo Final Integrador (TFI).

##  Descripción

Sistema diseñado para la gestión integral de una metalúrgica, permitiendo administrar clientes, productos, presupuestos, pagos y control de stock.

##  Tecnologías utilizadas

* Node.js
* Express
* React
* Base de datos (MySQL / SQLite)

##  Funcionalidades

* Gestión de clientes
* Gestión de productos
* Generación de presupuestos
* Registro de pagos
* Control de stock

##  Instalación y ejecución

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```

##  Notas

Las dependencias (`node_modules`) no están incluidas.
Ejecutar `npm install` antes de iniciar el proyecto.
